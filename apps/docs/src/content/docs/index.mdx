VisLab provides **17 interactive canvas widgets** for technical writing: CPU pipelines, cache hierarchies, OS schedulers, compilers, and algorithms.

## Embed paths

- **React / Astro** — `@vislab/react` with `VislabMount` or named exports
- **Static HTML** — `data-vislab` + `vislab-embed.min.js`
- **Jekyll** — `vislab-jekyll` gem includes

All widgets are declared once in `@vislab/registry` and consumed by every adapter.

## Status

VisLab is in active development — not yet published to npm. See the monorepo [README](https://github.com/mohitmishra786/vislab) and [roadmap](https://github.com/mohitmishra786/vislab/blob/main/TODOS.md).
