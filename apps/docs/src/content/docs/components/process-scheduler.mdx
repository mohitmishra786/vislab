Spawn processes and switch between **round-robin** and a simplified **CFS** (min vruntime) selection. Quantum is configurable.

## Props

| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `algorithm` | `"round-robin" \| "cfs"` | `round-robin` | Selection policy |
| `quantum` | `number` | `1000` | Time slice (sim ms, clamped 100–10000) |
| `themeName` | `string` | `dark-premium` | Theme id |

## Pedagogy notes

Not Linux CFS. Equal weight, no nice values, no sleepers. Use for intuition only.
