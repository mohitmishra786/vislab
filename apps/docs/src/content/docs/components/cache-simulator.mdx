## Props

| Prop | Type | Default | Description |

| `lineCount` | `number` | `8` | Memory lines shown (4–16) |
| `themeName` | `string` | `dark-premium` | Theme id |

## Pedagogy notes

Associativity, inclusivity, and multi-core coherence are **not** modeled. FIFO does not bump recency on hits; LRU does.
