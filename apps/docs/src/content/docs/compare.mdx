| Need | Best tool |
| ---- | --------- |
| Flowcharts / sequence from text | **Mermaid** |
| Freeform whiteboard sketch | **Excalidraw** |
| Data notebooks / custom D3 essays | **Observable** |
| Hosted runnable sandboxes | CodeSandbox / CodePen |
| **Systems temporal sims in your MDX blog** | **VisLab Widgets** |
| Full algorithm portal | VisuAlgo-class sites |

## Positioning

- **Mermaid** draws structure; VisLab simulates **time** (pipelines, latency, scheduling).
- **Excalidraw** is freeform; VisLab is a **catalog of curriculum widgets**.
- **Video** is heavy and hard to scrub pedagogically; VisLab is interactive and ~16–19 KB gzip for the embed kits.
- **Custom canvas per post** works once; VisLab gives registry + React/Jekyll adapters so you do not rebuild every time.

## Whitespace we own

1. Systems MDX components (CPU / OS / storage)
2. Latency-accurate teaching illustrations for eng blogs
3. Export-to-GIF for social while keeping live embeds on the article

## What we will not become

General diagramming, BI dashboards, or a closed teaching portal.
