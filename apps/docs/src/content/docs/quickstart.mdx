## Prerequisites

- Node.js **≥ 22.12**
- pnpm 10+

## Status: monorepo first

Until `@vislab/*` is published to npm, clone and build:

```bash
git clone https://github.com/mohitmishra786/vislab.git
cd vislab && pnpm install && pnpm run build
```

## React / Astro

```astro
---
import { StorageComparison } from "@vislab/react";
---
<figure>
  <StorageComparison client:visible />
  <figcaption>Relative storage-tier latency race.</figcaption>
</figure>
```

`client:visible` avoids hydrating off-screen islands (recommended).
